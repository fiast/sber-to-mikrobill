# Sber-to-MicroBill

Парсер уведомлений СберБизнес о поступлениях → TSV-реестр для импорта в MicroBill.

## Возможности
- Подключение к почте по IMAP
- Поиск писем об операциях по счёту
- Парсинг HTML-писем (quoted-printable)
- Извлечение: дата, сумма, плательщик, ИНН, лицевой счёт, договор, УПД
- Пропуск списаний и эквайринга
- Вывод TSV-реестра

## Установка
```bash
git clone ...
cd sber-to-mikrobill
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# отредактировать .env

### Вывод ключевых полей через awk

```bash
awk -F'\t' '
NR==1 {for(i=1;i<=NF;i++) header[i]=$i}
NR>1 {
    print "--- Запись " NR-1 " ---"
    for(i=1;i<=NF;i++) {
        if(header[i]=="Дата" || header[i]=="Сумма" || header[i]=="Плательщик" || header[i]=="ЛицевойСчет" || header[i]=="НазначениеПлатежа")
            printf "%-20s: %s\n", header[i], $i
    }
    print ""
}' microbill_reestr.tsv
```
